# Dataset samples

This folder contains a small visual sample from each dataset used in the study:

- [`TN5000_internal`](./TN5000_internal/) — internal development/evaluation examples
- [`TN3K_TNCD_external`](./TN3K_TNCD_external/) — independent external-validation examples

The repository includes only ten example images and compact manifests. It does not redistribute the complete datasets.
